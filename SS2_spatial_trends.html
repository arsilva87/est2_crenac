Trend analysis

A spatial trend model is defined as

\[Z(x_i) = \mu(x_i) + \epsilon(x_i)\] where the expected mean \(\mu(x_i)\) is a function of location \(x_i = (x, y)\); \(\epsilon(x_i)\) is a zero-mean error.

Parametric trend surfaces

  • \(g[\mu(x_i)] = \beta_0\)
  • \(g[\mu(x_i)] = \beta_0 + covariates\)
  • \(g[\mu(x_i)] = \beta_0 + \beta_1x + \beta_2y\)
  • \(g[\mu(x_i)] = \beta_0 + \beta_1x + \beta_2y + \beta_3x^2 + \beta_4y^2 + \beta_5xy\)
  • \(g[\mu(x_i)] = \beta_0 + h(x, y)\)

where \(g[\cdot]\) is called link function (e.g.log, for Poisson counts); \(h(\cdot)\) is a smooth function (e.g.splines)

Comparisons of models

Akaikes Information Criterion (the lower the better):

\[AIC = -2\log L(\mu) + 2k\] \(L(\mu)\) is the likelihood function of the fitted model, \(k\) is the number of parameters (\(\beta\)).

For nested models, a LRT - Likelihood Ratio Test can be applied

\[R_L = 2[\log L(\hat{\mu}) - \log L(\beta_0)] \sim \chi^2_{\nu}\]

Further trend surfaces

  • \(g[\mu(x_i)] = \beta_0 + S(x_i)\)
  • \(g[\mu(x_i)] = \beta_0 + S(x_i) + N_i\)

where \(S(x_i)\) is a spatial gaussian random effect with mean zero, variance \(\sigma^2\) and correlation \(\rho(d)\); \(N_i\) is a non-spatial random effect with variance \(\tau^2\) (nugget)

Generalized least squares

\[\hat{\beta} = (X'\Sigma^{-1}X)^{-1}X'\Sigma^{-1}z\]

\(\Sigma\) is the residual covariance matrix, which allows to model, for example, correlations of the form

\[\rho(d) = (1-\tau^2)\exp(-d/\phi)\]

Fitting a model with correlated errors

library(nlme)
model_3 = gls(abundance ~ x+y, data = abund,
