Objective

Learn how to retrieve data from vector data, check spatial autocorrelation, perform spatial interpolation in a R.O.I., visualize and save a map in a raster file using R.

Spatial autocorrelation

  • Morans I

\[I = \frac{n}{\sum_{i=1}^n (z_i - \bar{z})^2} \frac{\sum_{i=1}^n \sum_{j=1}^n w_{ij}(z_i - \bar{z})(z_j - \bar{z})}{\sum_{i=1}^n \sum_{j=1}^n w_{ij}}\]

\(z_i\) (and \(z_j\)) is a spatial observation, \(w_{ij}\) is a spatial weight for the link between \(i\) and \(j\).

Morans I test

\(H_0: I = E(I) = -1/(n-1)\) vs.\(H_1: I > E(I)\)

library(spdep)
wl = mat2listw(W) 
moran.test(abund$abundance, listw = wl)

                       method = "corr", zero.policy = TRUE)

Clusters and Hotspots

  • Clusters: observations with very similar neighbours
  • Hotspots: observations with very different neighbours

Those can be detected by breaking the global statistics (e.g.Morans I) into its n components:

\[I_i = \frac{n(z_i - \bar{z})\sum_{j=1}^n w_{ij}(z_j - \bar{z})}{\sum_{i=1}^n (z_i - \bar{z})^2}\]

Local Morans I

locI = localmoran(abund$abundance, wl2)
abund$quadr = factor(attr(locI, 'quadr')$mean,
                      levels = c('Low-Low', 'High-High',
                                 'Low-High', 'High-Low'))
par(mar = c(1,1,1,1))
spplot(abund, zcol = 'quadr')

Spatial interpolation

  • Non-geostatistical (IDW, Trend analysis - regression)
  • Geostatistical (Kriging)

Inverse Distance Weighting

\[\hat{z}(s_k) = \frac{\sum_{i=1}^n w(s_i)z(s_i)}{\sum_{i=1}^n w(s_i)}\]

where \(w(s_i) = ||s_k - s_i||^{-p}\), and usually \(p = 2\)

Prediction grid

north = subset(vistula, Name == 'northern_island')
gr = spsample(north, n = 5000, type = 'regular')
colnames(attr(gr, 'coords')) = c('x', 'y')
gridded(gr) = TRUE
crs(gr) = crs(north)
par(mar = c(1,1,1,1)); plot(north)
points(gr, cex = 0.5, col = 'blue')

IWD interpolation

colnames(attr(abund, 'coords')) = c('x', 'y')
library(gstat)
pred_idw = idw(abundance ~ 1, abund, newdata = gr, idp = 2)
## [inverse distance weighted interpolation]

Prediction map

r = raster(pred_idw)
par(mar = c(5,5,1,1))
plot(r)
